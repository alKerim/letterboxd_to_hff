# Quick Installation Guide

## Install the HFF Library Availability Extension

### Step 1: Open Chrome Extensions
1. Open Google Chrome
2. Navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right corner

### Step 2: Load the Extension
1. Click "Load unpacked" button
2. Select the `letterboxd_hff_chromeextension` folder
3. The extension should appear in your extensions list

### Step 3: Test the Extension
1. Navigate to any Letterboxd page (e.g., https://letterboxd.com/films/)
2. Click the extension icon in the Chrome toolbar
3. Click "Scan for HFF" to start checking availability
4. You should see green dots appear on film tiles that are available at HFF Munich library

### Step 4: Use the Extension
- **Green dots** indicate films available at HFF
- **Click the dots** to open the catalog entry in a new tab
- **Hover over dots** to see "Available at HFF Mediathek" tooltip

## Troubleshooting

### Extension Not Working?
- Make sure you're on a Letterboxd page (letterboxd.com)
- Click the extension icon and then "Scan for HFF"
- Check that the extension is enabled in `chrome://extensions/`

### No Green Dots?
- The extension only shows indicators for films actually available at HFF
- Check your internet connection
- The HFF WebOPAC system might be temporarily unavailable

### Performance Issues?
- The extension caches results, so subsequent visits will be faster
- If the page feels slow, try refreshing

## What the Extension Does

1. **Scans Letterboxd pages** for film tiles
2. **Extracts film titles and years** from the page
3. **Searches the HFF WebOPAC system** for each film
4. **Shows green indicators** on available films
5. **Provides clickable links** to catalog entries

## Privacy

- Only accesses Letterboxd and HFF WebOPAC
- No data sent to third-party servers
- Results cached locally in your browser
- No personal information collected
